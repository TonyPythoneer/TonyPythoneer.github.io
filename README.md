# 快速起手

ref: https://hexo.io/zh-cn/docs/writing.html

* 新頁面

```sh
$ hexo new page <title>
```

* 發文

```sh
$ hexo new post <title>
```

* 草稿

```sh
$ hexo new draft <title>
```

* 將草稿移動成發文

```
$ hexo publish post <title>
```

* 建立模版

```
$ hexo new photo <title>
```

在執行這行指令時，Hexo 會嘗試在 `scaffolds` 資料夾中找尋 `photo.md`，並根據其內容建立文章
